Nationalexpress.com Accessibility and Usability Evaluation

Accessibility

The website Nationalexpress.com has both advantages and weaknesses in matters of accessibility. Some options of website people with disabilities can easily use, although the site needs a substantial revision and additional tests to attract all categories of users. To date, the site can be used with external assistive technologies.

# Test Results
1 Page title Website has accessible page title which consists appropriate information about visited page.
2 Images ALT Attributes Not all images contain the alternative text. Some pictures have incomprehensible alternative text. Website is not adapted for viewing without images, because text merges with background and the website becomes completely inaccessible to all categories of users.
3 Appropriate use of tables Tables are not used for design of website
4 Headings All web pages do not contain <h1> headers, but <h2>, <h3> and <h4> headings are used 
5 Luminance contrast ratio The web site contains combinations of fonts and backgrounds which difficult to distinguish. Website is not adapted for viewing without images, because text merges with background
6 Resize the text Text on the website can be easily resized , and the text and images do not overlap. All blocks of website depending on font size are displayed correctly
7 Keyboard access and visual focus Website does not provide sufficient tab control navigation. Also, cursor is not highlighted, and it is impossible to distinguish where it is placed
8 Form, Label and Errors Form elements are accessible from the keyboard, but it cannot be tabbed in a certain sequence. The fields are not marked as required, but provide the appropriate error explanations. The automatic checker showed that forms are not completely accessible, and some fields do not have label attributes. Headings are used instead label attributes.
9 Login and signup forms Register and Login form appearance on the same page. Login form is available after Register form from keyboard
10 Removal of stylesheet and images Information appear in order and available for screen readers

Usability

The website seems quite usable, although some information on the website and services can be confusing for users and it could be useful to encourage more people for testing and address the issues to improve usability of the website

# Test Results
1 Site purposes Name of the website has an appropriate size and noticeable location. Site name clearly defines the purposes. Directly on the Home page a searching form is placed for checking requested transport and buying the tickets. Users can use searching form directly and do not waste time reading unnecessary information. All current offers clearly emphasized on the Home page
2 Communicating Information About Your Company The website contains menu item “About us”, but some nested menu items do not match the name of the parent menu item and can confuse the users
3 Content Writing Important sections are well emphasized. The combination of clear text and icons allow users to avoid confusion on the website. The webpages do not contain redundant information. The well-highlighted headers make it easier to navigate on the pages.
4 Revealing Content Through Examples Each example on the webpages consists direct link, which goes directly to the detailed information
5 Archives and Accessing Past Content The website does not provide a section of recent viewed destinations and type of transport
6 Links Links on the website have understandable text and do not confuse users. However, the links are not highlighted with color which could show visited and unvisited links
7 Navigation Navigation of the website is clear and comprehensive, despite the nested menu items
8 Search A search form contains several fields to fill, and it is available on the majority of the pages. Also, the form “Ask the question” is displayed in the top right corner and can be confused with the search form. Text fields of the forms are long enough to type there requested information
9 Tools and Task Shortcuts The website has drop list menu to change the language of the website
10 Graphics and Animation Presented images illustrate real content and icons used on the website are clear. Animation is used on the website to present current offers and discounts
11 Graphic Design Font styles are limited, but some text has low contrast and merges with background. The website displayed on the screen with 1024x768 resolution with horizontal scroll
12 URLs URL of the home page is simple and memorable
13 Popup Windows and Staging Pages Pop-up are not used on the website, but it does not distract usability of the website. Routing pages mechanism is not used on the website
14 Advertising The website provides offers and discount advertising which tends to distract users’ attention
15 Page Reload and Refresh The website does not reload and refresh the content automatically
16 Customization The website saves customizable parameters of users for subsequent visits
17 Dates and Times Dates and times definitions are provided in legible form. Users can see that information updated on the website

Universal Design Tests

# Test Results
1 Simplicity The content of the website looks simple and understandable. All essential information and options attracts the eye and makes information accessible
2 Central Layout The website is positioned centrally
3 Fewer Columns 2 columns design is used
4 Separate top section Title and navigation are clearly separated from the rest of the webpage
5 Solid areas of screen real-estate All areas of website can be distinct clearly from each other, because each block is designed as a sticker
6 Simple navigation The website has a clear navigation, in spite of 2 levels
7 Bold logos Bold logo is used on the website. It is clear and makes the first impression about the website and content
8 Bigger text Some bright colours emphasize important data and forms; however, some text merge with and background
9 Bold text introductions All sections are provided with bold captions
10 Strong colors Some bright colours emphasize important data and forms, however some text merge with and background
11 Rich surfaces 3D effects are commonly used on the website
12 Gradients Almost every section includes gradient background
13 Reflections Some images and buttons include reflection effects
14 Cure icons Icons on the website have good-looking design, recognizable and have a clear meaning
15 Star flashes Some sections of the website use star flashes
