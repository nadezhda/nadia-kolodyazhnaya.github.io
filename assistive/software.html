Software Evaluation

Two desktop software applications Total Commander and Skype were chosen for the accessibility and usability evaluation. These applications were tested with Microsoft Windows 7 Operation System (OS) 64 bit and following external accessibility tools were used for examination:

  • NVDA software for blind and vision impaired people
  • Virtual Magnifying Glass application for people with low vision and dyslexia
  • DSpeech software which converts audio into transcript

For accessibility both desktop software applications were examined using Guidelines for Application Software Accessibility of The Centre for Excellence in Universal Design (CEUD). The resource includes guidelines and standards for assistive software, and universal design and it contains accessibility test cases for struggling people.

The usability tests are relied on “10 Usability Heuristics for User Interface Design” guideline written by Jakob Nielsen who has valuable experience in interface evaluation and “guiding critical design decisions”.
