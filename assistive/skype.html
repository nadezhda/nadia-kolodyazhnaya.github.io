Skype Accessibility and Usability Evaluation

Skype is one of the most usable applications, which use Voice over Internet Protocol (VoIP) services and provides communication between the peers using the voice, video and text messages. Skype also supports the exchange of media files and video conferencing between several participants. Nowadays, Skype is available for different platforms and devices. The application continues to develop and encourages new users. Unfortunately, it seems that recent changes have not improved interface of the programme.

Accessibility

The application works adequately with Windows and external assistive tools, though build-in accessibility tools were not found in the programme. Surprisingly, desktop version of Skype contains Accessibility option which enables “Accessibility Mode”, but no accessibility improvements were found. Skype has intuitive graphical interface, but menu bar contains too many options which can be confusing. The usage of Skype tends to be difficult for people with hearing impairments because programme does not provide transcript and works badly with external text to speech tools.

# Test Results
1 Built in Accessibility Tools Access Application works well with build-in assistive tools without affection on functionality
2 Application works with External Assistive Technologies Skype works well with external assistive technologies such as NVDA and VirtualMagnifyingGlass
3 Keyboard access methods Not all elements of the application are available from the keyboard and can be tabbed. Some options are available just using the mouse pointer
4 Label for objects, fields and controls All icons of the programme contain explaining tooltip
5 Textual information for screen reader All buttons, content and fields are available for NVDA screen reader tools.
6 Change font, color and contrast The application does not contain any settings for colour or contrast changing.
7 Save user preferences for style and zoom Skype does not have any style and zoom customizations
8 Audio alerts Application includes various build-in audio notifications
9 Alternatives for Video / Animation Application presents real time video and audio content. Transcripts and captions are not provided. Also, internal programme DSpeech does not work with Skype
10 Flashing affects The new versions of the programme include advertising blocks which have some flashing affects
11 Simplicity The interface of the programme intuitive and contains simple and clear descriptions and labels. All icons in the programme have understandable tooltips. However, menu bar of the application contains various items which tend to confuse users

Usability

Desktop version of Skype was evaluated in usability and results are shown in Table IV. It seems adequately usa-ble and intuitive application, though some options of the programme can confuse people. All users’ actions and events are provided with reasonable and clear notifications; programme has user-friendly interface and customizable options. However, some unexpected errors can be appeared, and some additional advertising tend to distract users’ attention.

# Test Results
1 Visibility of system status All the processes in the programme accompanied by notifying users
2 Match between system and the real world The language used in the programme is clear and understandable
3 User control and freedom The program interacts with the user through notification messages. Some operations cannot be canceled, for example, sent messages cannot be removed.
4 Consistency and standards The programme contains set of the messages and events which tends to be understandable for users. However, some events are not obvious, for instance users cannot understand exactly messages were delivered, read or delayed
5 Error prevention User's actions are accompanied by messages. If the error was detected or it is a poor connection, the program provides respective notification. Sometimes programme provides dialog window to assess the quality of connection and leave feedback.
6 Recognition rather than recall The programme allows operating of contacts, creating new groups, change the status of activity
7 Flexibility and efficiency of use The programme includes set of the quick buttons and option, which can help speed up the user experience
8 Aesthetic and minimalist design Interface of the programme is clear, the dialogs do not contain irrelevant or unnecessary information, but some advertising, which was added recently, can distract the attention of users
9 Help users recognize, diagnose, and recover from errors Error messages do not include any codes, but some unexpected errors can be appeared and close the programme
10 Help and documentation Skype provides documentation which is available from the menu bar item “Help”
