Assistive Tools Evaluation

Nowadays, users with different types of disabilities are forced to use a variety of tools to make applications and websites more accessible. Sometimes, people install and operate several programs which can be confusing for users. Also, each browser has own instruments.

Two assistive tools ATbar cross-browser toolbar and desktop application "Access Tools" were tested and analysed.
