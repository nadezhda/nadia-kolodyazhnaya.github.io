ATbar

ATbar is an open source web tool which provides a variety of options to make websites more accessible for dif-ferent target groups regardless of type of browser. It can significantly improve accessibility of websites for people with different disabilities, and also, for foreign and average users. The programme contains many different options which benefit not just users, but developers as well. Developers provide several different versions of the application for any browser, Windows OS or it can be built-in directly in a particular site. Also, users have an option to create own ATbar by choosing essential instruments.

Visually, the application is a set of buttons that trigger the appropriate plugins to process websites in more acces-sible way. ATbar consists a large number of options that can meet the needs of majority of struggling users.

The evaluation showed ATbar is easy to use and tends to be helpful for the majority of users. It can be straight-forward implemented into the browser by an average user as well as developers can built-in it into a particular web-site if it is written in php programming language. Using ATbar can help to avoid the use of various external assistive programs and it perfectly works with evaluated above websites. However, some lacks of the programme were discovered. Unfortunately, the options for deaf people and users with hearing impairments were not discovered. Also, users should have ability to change location of pop-up win-dows, because they can overlap on each other, for instance word prediction tooltip and style changing pop-up win-dow. As for text to speech recognition, probably it will be beneficial to add some setting to customize the speed of speech.

# Tools Description
1 Magnification AT bar allow increasing and decreasing font size on the webpage respectively. According to instructions on ATbar website, users can resize the content of the websites using a browser zoom. The option tends to help to older people and people with low vision
2 Font settings ATbar presents the options to modify font type and line space. These settings can play a crucial role for people with dyslexia and other reading difficulties. Alse it is suggested using additional font settings such as a space between characters and paragraphs.
3 Text to speech The option can be useful for blind people or those with low vision and reading difficulties. In pop-up window reader voice can be chosen. The text remains selected when activating the text to speech option, but it tend to be better remove selection and apply highlighting of words during the reading.
4 Readability This option present textual version of the web page without clutter. Readability option allows printing, sharing the page and also reading page later. In addition, this option can be useful for people with reading difficulties and those with poor Internet connection.
5 Spellchecker ATbar activates the option of the spell checker during the writing in the text forms. It can be useful for the different target groups as people with dyslexia, poor spelling and foreign users.
6 Dictionary This option also can be useful for users who do not familiar with a particular language. User can select unfamiliar word and press the button "Dictionary", after that the pop-up window shows the definition of an unknown word. Dictionary definition provides a full range of explanation the word, but transcription and pronunciation of words is missed..
7 Word prediction This option can be valuable for users with physical disabilities, dyslexia, and foreign users. The application activates word prediction option which is available for text fields. The option available from the keyboard using three-key combinations. However, it could tend not to be achievable for users with physical disabilities.
8 Style changing AT bar provides an opportunity to change text and link color and color of the background.
9 Color overlay It is a valuable option for people with dyslexia and reading difficulties who have difficulties with eye-coordination and focusing. Option allows masking the web page thereby decrease the contrast.
