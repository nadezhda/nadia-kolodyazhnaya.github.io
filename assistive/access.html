Accessibility of the Current Website

The website was created based on previous website accessibilty evaluations as an example and complies with the requirements Web Content Accessibility Guidelines (WCAG) 2.0.

The website was tested using Content Accessibility Guidelines (WCAG) 2.0, which is official W3C Recommendation, and Accessibility Initiative Working Draft which contains a description of tools and methods for evaluation. For examination of websites, manual tests were conducted, and next instruments have been used for automatic checking:

  • Web Developer 1.2.5 for FireFox browser (FF toolbar)
  • Fangs 1.0.8 add-on for FireFox browser which displays text version of screen reader tool
  • WAVE (Web Accessibility Evaluation Tool)
  • Juicy Studio Accessibility Toolbar for FireFox
  • IE WAT
# Test Results
1 Page title Website has accessible page title, where new information appears at the beginning of page title.
2 Images ALT Attributes All images contain an appropriate alternative text
3 Appropriate use of tables Table are not used for design of the website, but used to present particular information
4 Headings Hierarchy of headings is used on the website, where <h1> heading is the name of the website
5 Luminance contrast ratio The web site contains clear contrast between fonts and backgrounds
6 Resize the text Text on the website can be easily resized with additional assistive tools such as ATbar
7 Keyboard access and visual focus The website provides key navigation and each element can be tabbed in certain sequence. Moreover, menu of the wibesite is tabbed primarly.
8 Form, Label and Errors Current website does not contain any forms
9 Login and signup forms Current website does not contain login and signup forms
10 Removal of stylesheet and images Information from website is available for external screen readers
