Total Commander Accessibility and Usability Evaluation

Total Commander is a desktop file manager application which is compatible with different OS. Total Commander contains a number of options which makes easier work with folders and files. The software application includes some features as a build-in file FTP client, interface which is totally accessible for keyboard, file compare and accessible navigation. Total Commander supports build-in options as well as proprietary Windows features and shortcuts.

Accessibility

Total commander does not include build-in assistive instruments, but it works well with Windows and external assistive tools. Also, programme supports changing font setting and it can be beneficial for those with dyslexia and low vision. The interface of the programme is comprehensive and every icon has effective tooltips. In addition, Total Commander maintains the keyboard access and shortcuts.

# Test Results
1 Built in Accessibility Tools Access Application works well with Windows build-in assistive tools without affection on functionality
2 Application works with External Assistive Technologies Total commander works well with external assistive technologies such as NVDA and VirtualMagnifyingGlass
3 Keyboard access methods Application completely available from the keyboard and support proprietary and Windows shortcuts
4 Label for objects, fields and controls All icons of the programme contain explaining tooltip
5 Textual information for screen reader Total commander allows changing a font size and type as well as customize colour of text and background
6 Change font, color and contrast The application saves user’s preferenced
7 Save user preferences for style and zoom Skype does not have any style and zoom customizations
8 Audio alerts The programme uses audio alerts of OS. All alerts can be intuitively understandable for people with hearing impairments and do not require additional transcript
9 Flashing affects The programme does not have any flashing affects
10 Simplicity The interface of the programme intuitive and contains simple and clear commands, descriptions and labels. All icons in the programme have understandable tooltips
11 Documentation Documentation is available from Total Commander window

Usability

Total Commander makes an impression as user friendly and intuitive application. The main window and additional dialog windows of programme contain clear notifications and comprehensive controls. Quick buttons of the programme tend to improve speed of work with application.

# Test Results
1 Visibility of system status All the processes in the programme accompanied by notifying users
2 Match between system and the real world The language used in the programme is clear and understandable
3 User control and freedom The program interacts with the user through dialog windows. The user has the option to cancel the operation.
4 Consistency and standards The control elements of software application corresponds to interface and functionality Windows and can be familiar to user
5 Error prevention User's actions are accompanied by confirmation windows. If the error is detected, the program provides respective notification
6 Recognition rather than recall The programme interface consists 2 panels, which provide convenient work with folders and files. The programme also allows creating tabs with a specific location in the file system
7 Flexibility and efficiency of use The programme includes set of the quick buttons, which can help speed up the user experience
8 Aesthetic and minimalist design Interface of the programme is clear, dialogs do not contain irrelevant or unnecessary information
9 Help users recognize, diagnose, and recover from errors Error messages do not include any codes, but just comprehensive text, which clearly indicate the problem
10 Help and documentation Total Commander provides documentation which is available from separate button “Reference”
