Web 2.0 Evaluation

Two websites which provides booking and selling services were chosen for the evaluation. Both websites were tested using Content Accessibility Guidelines (WCAG) 2.0, which is official W3C Recommendation, and Accessibility Initiative Working Draft which contains a description of tools and methods for evaluation. For examination of websites, manual tests were conducted, and next instruments have been used for automatic checking:

  • Web Developer 1.2.5 for FireFox browser (FF toolbar)
  • Fangs 1.0.8 add-on for FireFox browser which displays text version of screen reader tool
  • WAVE (Web Accessibility Evaluation Tool)
  • Juicy Studio Accessibility Toolbar for FireFox
  • IE WAT

For usability evaluation of the websites, the guidelines "113 Design Guidelines for Homepage Usability" of Jakob Nielsen was used. As mentioned in the resource, the guideline is based on 14-years of experience and looks reli-able for examination of websites.

As a guideline for designing to standards evaluation, tutorial "Web 2.0 Design Guide" was used. This guideline highlights some aspects of design specific to Web 2.0.

For the websites, browser and operating System Compatibility also was examined using Cross Browser Testing Tools and HTML/CSS code was validate using W3C service.
