Usability of the Current Website

For usability evaluation, the guidelines "113 Design Guidelines for Homepage Usability" of Jakob Nielsen was used.

The website was created following guidelines mentioned above. However, the flexible design template was used to satisfy as possible the requirements of the resolution and size of the user's screens. The website includes simple content and not overloaded with pictures, though it contains numerous tables which presents the results of evaluations and which can be difficult to read. However,there was an attempt to develop the tables so that information is easily read and understood.

Navigation has been designed so that the user can be as easy as possible to use the site. Thinking about the people managing the web site only with the keyboard, the keyboard shortcut was introduced to the main menu and all items can be tabbed in certain sequence with clear highlighting.
