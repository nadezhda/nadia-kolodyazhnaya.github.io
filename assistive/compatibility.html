Browser and OS compatibility

Browser and OS Compatibility was checked using Web Developer Tools for ForeFox browser. Also, appearance of the website was tested in different browsers such as IE, FireFox and Opera. Unfortunately, the website is not designed for display on the screens of mobile smartphones, but it is watched on the tablets with minimum resolution 600x800px and with portrait and landscape orientation. The Cross Browser Testing Tools could not be used for evaluation since the website does not have particular host.
