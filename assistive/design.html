Designing of standards of the Current Website

As a guideline for designing to standards evaluation, tutorial "Web 2.0 Design Guide" was used. This guideline highlights some aspects of design specific to Web 2.0.

Universal Design Tests

# Test Results
1 Simplicity The content of the website was devided on the sections using a simple presentation without additonal clutter. May be it will be better to add additional features which will highlight essential information and attract the eyes of users.
2 Central Layout The website uses flexible design template
3 Fewer Columns 2 columns design is used (sidebar and content)
4 Separate top section Title and navigation are clearly separated from the rest of the webpage
5 Solid areas of screen real-estate All areas of website can be distinct clearly from each other
6 Simple navigation The website has a clear navigation which presented in the header and sidebar of the website
7 Bold logos Heading of the website uses the bold text and logo is represented as picture with link on the Home Page
8 Bigger text Titles are clearly highlited using the bigger text and dashed lines
9 Bold text introductions As mentioned above, all sections are provided with bold headings
10 Strong colors Some bright colours emphasize menu item where located users on the website, colors do not merge with background
11 Rich surfaces 3D effects are not used on the website
12 Gradients Gradients are used on the website as a background for header, footer, sidebar and tables' captions
13 Reflections Some images and buttons include reflection effects
14 Cure icons Icons are not used on the website
15 Star flashes The website does not contain star flashes
